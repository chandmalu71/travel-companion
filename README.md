# Travel Companion

AI-powered travel planning and companion app. Built with React, TypeScript, and AWS.

## Status

Project initialized. Specs pending.

## Technology Stack

- **Frontend:** React 18 + TypeScript
- **Infrastructure:** AWS (S3, CloudFront, Lambda, DynamoDB, Cognito) — eu-west-1
- **CI/CD:** GitHub Actions
- **Testing:** Jest + Playwright

## Getting Started

```bash
npm install
npm start
```
